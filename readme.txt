== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go into the widget page: `/wp-admin/widgets.php`
1. Drag the widget named CakeMail to the sidebar
1. Log into the widget to your CakeMail account (or create a free account on [CakeMail](http://cakemail.com/ "CakeMail"))
1. Arrage the order and the name of the fields
1. Start collecting subscribers
